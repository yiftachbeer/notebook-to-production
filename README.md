# DepthSegTo3D

Turn a driving image into a colored 3D point cloud: run a monocular **depth**
model and a semantic **segmentation** model, then back-project every pixel into
3D using the camera's real calibration.

## Run it

Open `DepthSegTo3D.ipynb` in Google Colab with a GPU runtime and run all cells.
The data is downloaded by the notebook.
